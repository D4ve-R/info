✨ サブセット

はじめに#

🎩 ほとんどの BIM アプリケーションでは、ユーザーが要素の上にマウスを移動させたり、選択したりすると、要素がハイライトされます。IFC.js も例外ではありません。このチュートリアルでは、これを実現する方法を見ていきます。

👩‍🏫 前回のでは、RaycasterでレイをキャストしたオブジェクトのIDを取得する方法を見ました。これらのIDが得られたので、幾何学的なサブセットを使用してそれらの要素を強調することができます。
🧱

幾何学的なサブセット？

幾何学的サブセットは、モデル内のアイテムの任意のグループです。例えば、すべてのドア、すべての1階要素、または与えられた条件を満たすすべての要素などです。

インポートの依存性#

🏀 アイテムを強調するためには、materialが必要です。この例では、Three のコアライブラリからインポートしたMeshLambertmaterialを使用しています。

import { MeshLambertMaterial } from "three";

今回は Raycaster を使用するので、これらの依存関係も必要ですし、最適なパフォーマンスを求めるならthree-mesh-bvhも必要です。

その方法とは#

ハイライト素材#

🌀 まず最初に、ハイライトマテリアルを作成します。設定を変更して、マテリアルをきれいに見せましょう。プロのアドバイス: depthTest=false を使用すると、どの視点からでもオブジェクトが見えるようになります。

// サブセット素材の作成
const preselectMat = new MeshLambertMaterial({
transparent: true,
opacity: 0.6,
color: 0xff88ff,
depthTest: false,
});

🎭 幾何学的なサブセットは、その素材によって一意に識別されることに注意してください。

  • 材料 A を使用した壁でサブセットを作成した後、同じ材料 A を使用した別の壁のサブセットを作成しようとすると、2 番目の壁が最初の壁のサブセットに追加されます。

  • 材料 A を使用して壁でサブセットを作成し、次に材料 B を使用して同じ壁の別のサブセットを作成しようとすると、2 つの独立したサブセットが作成されます。

💡
同じ素材の2つのインスタンスを使って、同じ外観の2つのサブセットを作ることができます。

シングルサブセット#

🐭createSubsetを使えば、ユーザーがホバーしたときにハイライト効果を出すことができます。

const ifc = ifcLoader.ifcManager;
// 前のセレクションへの参照
let preselectModel = { id: -1 };
function highlight(event, material, model) {
const found = cast(event)[0];
if (found) {
// モデルIDの取得
model.id = found.object.modelID;
// Express IDを取得する
const index = found.faceIndex;
const geometry = found.object.geometry;
const id = ifc.getExpressId(geometry, index);
// サブセットの作成
ifcLoader.ifcManager.createSubset({
modelID: model.id,
ids: [id],
material: material,
scene: scene,
removePrevious: true,
});
} else {
// Removes previous highlight
ifc.removeSubset(model.id, material);
}
}
window.onmousemove = (event) => highlight(event, preselectMat, preselectModel);

👀 いくつかの興味深い点があります。

  • cast()の実装については、前回で紹介しました。

  • 変数currentModelは、選択されたモデルの参照を格納するために使用されます。このようにして、ユーザーがオブジェクトにカーソルを合わせていないときに、removeSubsetで前のサブセットを削除することができます。

  • サブセットを作成するアイテムの ID は、配列として与えられなければなりません（単一の ID であっても）。

複数のサブセット#

💅💅💅 複数のサブセットを扱うのは、単一のサブセットを扱うのと同じくらい簡単です。新しい材料を作成し、同じ機能を持つ新しいサブセットを作成し、そのサブセットの作成をイベントに関連付けるだけでよいのです。

この例では、ダブルクリックにバインドすることで、オブジェクトが選択されたときにハイライトされる効果をシミュレートします。

const selectMat = new MeshLambertMaterial({
transparent: true,
opacity: 0.6,
color: 0xff00ff,
depthTest: false,
});
const selectModel = { id: -1 };
window.ondblclick = (event) => highlight(event, selectMat, selectModel);

ジオメトリの抽出#

👩‍🏫 ジオメトリサブセットを作成し、ハイライトマテリアルを指定しない場合、サブセットにはオリジナルマテリアルが使用されます。

💡
これにより、例えば、BIMモデルの1階部分のすべての要素で幾何学的なサブセットを作成し、それ以外の要素を隠すことができます。

🌚 次の例では、読み込んだ IFC モデルのコピーに透明なマテリアルを適用し、マウスをアイテムの上に置いたときに、オリジナルのマテリアルを持つサブセットを作成します。これには、前とほぼ同じコードを使用します。

ifcLoader.load("../../IFC/01.ifc", (ifcModel) => {
ifcModel.visible = false;
const modelCopy = new Mesh(
ifcModel.geometry,
new MeshLambertMaterial({
transparent: true,
opacity: 0.1,
color: 0x77aaff,
})
);
scene.add(ifcModel);
scene.add(modelCopy);
});
// ...
window.onmousemove = (event) => highlight(event, undefined, highlightModel);
