简介#

👩‍🏫 通过我们在其他教程中看到的内容，我们已经知道如何在 3D 中选择元素，访问它们的属性，并使用子集来突出它们，子集是整个模型的一部分。

🚪🧱 然而，在许多BIM 应用中，也有可能隐藏和隔离元素。一个常见的用例是隐藏所有不符合某个过滤器的元素，或者只显示属于建筑某层的对象。

🍰 当然，使用IFC.js 这是件小事。在这个例子中，我们要按类别创建过滤器，这样用户就可以用复选框显示或隐藏项目。

与其他教程一样，你可以找到完整的例子这里。

如何做到这一点#

获取类别的子集#

👩‍🏫 IFC.js 的可见性是基于子集操作的。这使得复杂的可视化能够以最小的内存占用来创建。我们要做的第一件事是定义哪些IFC 类别**我们将允许用户显示或隐藏。

📋📋📋 为了节省内存，IFC.js 中的类别被定义为数字常数。因此，让我们创建一个对象，将这些常量的名称映射到它们的数字值，并创建一个函数来检索它们。

import {
IFCWALLSTANDARDCASE,
IFCSLAB,
IFCDOOR,
IFCWINDOW,
IFCFURNISHINGELEMENT,
IFCMEMBER,
IFCPLATE,
} from "web-ifc";
// 类别名称列表
const categories = {
IFCWALLSTANDARDCASE,
IFCSLAB,
IFCFURNISHINGELEMENT,
IFCDOOR,
IFCWINDOW,
IFCPLATE,
IFCMEMBER,
};
// 获取一个类别的名称
function getName(category) {
const names = Object.keys(categories);
return names.find((name) => categories[name] === category);
}

🏠⏩🚪 现在让我们创建几个函数，以获得属于一个类别的元素的所有 ID，并用这些 ID 创建一个子集。

💡
记住，许多IFC.js函数是异步的，所以我们要使用async和await。

👩‍🏫 你也可以使用removeFromSubset()从一个子集中删除一个项目（例如，隐藏一个项目）。如果你把它和createSubset()结合起来，并且removePrevious = false，你就可以完全控制哪些内容被添加到哪个子集以及它的可见性。

// 获取一个特定类别的所有项目的ID
async function getAll(category) {
const manager = ifcLoader.ifcManager;
return manager.getAllItemsOfType(0, category, false);
}
// 创建一个新的子集，包含一个类别的所有元素
async function newSubsetOfType(category) {
const ids = await getAll(category);
return ifcLoader.ifcManager.createSubset({
modelID: 0,
scene,
ids,
removePrevious: true,
customID: category.toString(),
});
}

将 GUI 与可见性结合起来#

✨ 现在只剩下将每个复选框的点击事件与相应类别的可见性联系起来，这样，当复选框被激活时，类别就变得可见，而当它被停用时，类别就消失了。

🎨👨‍🎨 这里值得注意的是，为了使代码更加简洁，我们给每个复选框一个 ID，其名称与它所链接的类别相同。

// 存储已创建的子集
const subsets = {};
async function setupAllCategories() {
const allCategories = Object.values(categories);
for (let i = 0; i < allCategories.length; i++) {
const category = allCategories[i];
await setupCategory(category);
}
}
// 创建一个新的子集并配置复选框
async function setupCategory(category) {
subsets[category] = await newSubsetOfType(category);
setupCheckBox(category);
}
// 设置复选框事件来隐藏/显示元素
function setupCheckBox(category) {
const name = getName(category);
const checkBox = document.getElementById(name);
checkBox.addEventListener("change", (event) => {
const checked = event.target.checked;
const subset = subsets[category];
if (checked) scene.add(subset);
else subset.removeFromParent();
});
}
