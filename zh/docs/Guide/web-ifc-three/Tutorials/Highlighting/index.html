✨ 子集

简介#

🎩 在几乎所有的 BIM 应用程序中，当用户将鼠标移到元素上或选择它们时，元素会被高亮显示。IFC.js 也不例外，在本教程中我们将看到如何实现这一点。

👩‍🏫 之前我们已经看到了如何通过Raycaster来获得我们投射射线的对象的ID。现在我们有了这些ID，就可以使用几何子集来突出显示这些元素。
🧱

几何子集？

几何子集是模型中的任何一组项目。例如：所有的门、所有的底层元素或所有满足某个条件的元素。

进口依赖性#

⚽ 我们需要一个material来突出这些项目。你可以选择任何你喜欢的材料；在这个例子中，我们将使用一个MeshLambertmaterial，我们将从 Three 的核心库中导入。

import { MeshLambertMaterial } from "three";

我们将使用 Raycaster，所以你还需要这些依赖项，以及three-mesh-bvh，如果我们想获得最佳性能。

如何做到这一点#

亮点材料#

🌀 第一件事是创建高光材料.玩玩配置，让材料看起来很好! 专业提示：你可以使用depthTest=false，这样物体从任何视角都可以看到。

// 创建子集材料
const preselectMat = new MeshLambertMaterial({
transparent: true,
opacity: 0.6,
color: 0xff88ff,
depthTest: false,
});

☝ 请注意，几何子集是由它们的材料**独特地识别的。

  • 如果你用材料 A 创建了一个墙的子集，然后试图用同样的材料 A 创建另一个墙的子集，第二个墙将被添加到第一个墙的子集中。

  • 如果你用材料 A 创建了一个墙的子集，然后试图用材料 B 创建同一墙的另一个子集，你将创建两个独立的子集。

💡
You can have two separate subsets with the same appearance using two instances of the same material.

单一子集#

🐭 我们可以用createSubset来创建用户悬停时的高亮效果。

const ifc = ifcLoader.ifcManager;
// Reference to the previous selection
let preselectModel = { id: -1 };
function highlight(event, material, model) {
const found = cast(event)[0];
if (found) {
// 获取模型ID
model.id = found.object.modelID;
// 获取快递ID
const index = found.faceIndex;
const geometry = found.object.geometry;
const id = ifc.getExpressId(geometry, index);
// 创建子集
ifcLoader.ifcManager.createSubset({
modelID: model.id,
ids: [id],
material: material,
scene: scene,
removePrevious: true,
});
} else {
// 移除之前的高亮部分
ifc.removeSubset(model.id, material);
}
}
window.onmousemove = (event) => highlight(event, preselectMat, preselectModel);

👀 有几个有趣的地方需要注意。

  • cast()的实现已经展示了前面的。

  • 变量currentModel用于存储所选模型的引用；这样，当用户不在悬停一个对象时，我们就能用removeSubset删除之前的子集。

  • 要创建子集的项目的 ID 必须以**数组的形式给出（即使它是一个单一的 ID）。

多个子集#

💅💅💅 使用多个子集和使用单个子集一样简单。我们只需要创建一个新的材料，用同样的功能创建一个新的子集，并将该子集的创建与一个事件相关联。

在这个例子中，我们要把它绑定到双击，以模拟对象被选中时的高亮效果。

const selectMat = new MeshLambertMaterial({
transparent: true,
opacity: 0.6,
color: 0xff00ff,
depthTest: false,
});
const selectModel = { id: -1 };
window.ondblclick = (event) => highlight(event, selectMat, selectModel);

提取几何图形#

👩‍🏫 如果你创建了一个几何体子集，但没有指定高光材料，子集将有**的原始材料。

💡
例如，这将允许用BIM模型的所有底层元素创建一个几何子集，并隐藏其余部分。

🌚 在下一个例子中，我们将把透明材料应用于加载 IFC 模型的副本，当鼠标悬停在一个项目上时，用原始材料创建一个子集。为此，我们将使用与之前几乎相同的代码。

ifcLoader.load("../../IFC/01.ifc", (ifcModel) => {
ifcModel.visible = false;
const modelCopy = new Mesh(
ifcModel.geometry,
new MeshLambertMaterial({
transparent: true,
opacity: 0.1,
color: 0x77aaff,
})
);
scene.add(ifcModel);
scene.add(modelCopy);
});
// ...
window.onmousemove = (event) => highlight(event, undefined, highlightModel);
